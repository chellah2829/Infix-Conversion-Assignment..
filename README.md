**Description**

This project is a java program that converts an infix expression into postfix and prefix using a stack data structure 

**Workflow**

The program reads an infix expression processes it using a stack , and produces postfix and prefix outputs.

**Files Included**

- InfixConversion.java (Java source code)
- Assignment.docx (Contains workflow, pseudocode, and explanation) # **click  then press "view Raw" or "Download" button**

  **Sample Input**
  
  A+B*(C-D)

  **Sample Output**
  
  Postfix: ABCD-+
  Prefix: +AB-C
